# 🌐 Responsive Landing Page

## 📌 Project Description
This project is a responsive and interactive landing page developed as part of my Web Development Internship. The main objective of this project is to design a visually appealing and user-friendly interface with dynamic navigation behavior.

The landing page includes a fixed navigation bar that remains visible across all sections, along with interactive features such as scroll-based styling, active menu highlighting, and smooth transitions. The design focuses on modern UI principles to ensure clarity, responsiveness, and enhanced user experience.

---

## 🚀 Features
- Fixed navigation bar accessible on all sections  
- Navigation bar dynamically changes style on scroll  
- Active section highlighting in navigation menu  
- Smooth scrolling between sections  
- Interactive hover effects on menu items  
- Fully responsive layout for different screen sizes  
- Modern UI with gradient backgrounds and glassmorphism  
- Scroll-triggered animations for better visual engagement  

---

## 🛠️ Technologies Used
- HTML5 for structure  
- CSS3 for styling and responsive design  
- JavaScript for interactivity and dynamic behavior
